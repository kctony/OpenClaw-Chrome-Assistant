Privacy Policy

Last Updated: February 26, 2026

The OpenClaw Chrome Assistant extension is designed with privacy as a core principle. This policy explains how we handle data.

1. Data Collection

This extension does not collect, store, or transmit any personal data to servers owned or operated by the developers. All processing happens locally on your machine or via the OpenClaw Gateway you provide.

2. Data Transmission

Any data extracted from web pages (such as text, images, or URLs) is sent exclusively to your self-hosted or user-configured OpenClaw Gateway URL. This communication occurs via secure API calls initiated by your explicit configuration.

3. Permissions Usage

  • storage: Used solely to save your local configurations, such as your Gateway URL and Authentication Token.
  • activeTab: Used only when you interact with the extension to read the content of the currently active tab for processing.

4. Third-Party Access

No data handled by this extension is shared with, sold to, or accessible by third parties. You have full control over the data flow.

5. Contact

If you have questions about this privacy policy, please contact us via the GitHub repository or the support channel provided in the extension store.
